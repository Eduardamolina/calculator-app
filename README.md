# 🧮 Calculadora Web

Uma calculadora simples desenvolvida para praticar conceitos fundamentais de desenvolvimento web utilizando HTML, CSS e JavaScript.

## 📋 Sobre o projeto

Este projeto foi criado com o objetivo de aplicar conhecimentos básicos de:

* Estruturação de páginas com HTML;
* Estilização utilizando CSS;
* Manipulação do DOM e lógica de programação com JavaScript;
* Organização de arquivos em projetos front-end.

A aplicação realiza operações matemáticas básicas de forma simples e intuitiva.

## 🚀 Funcionalidades

* Adição
* Subtração
* Multiplicação
* Divisão
* Limpeza do visor
* Exibição do resultado em tempo real

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo de aprendizado

Este projeto foi desenvolvido como parte do meu processo de aprendizado em desenvolvimento web, com foco na prática de conceitos fundamentais do front-end e no uso do Git e GitHub para versionamento de código.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Manipulação de eventos com JavaScript;
* Atualização dinâmica do conteúdo da página;
* Organização de código;
* Versionamento utilizando Git;
* Publicação de projetos no GitHub.
